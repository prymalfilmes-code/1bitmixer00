# 1BitMixer00

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/prymalfilmes-code/1bitmixer00.git
   cd 1bitmixer00
   ```

2. Install the required dependencies:
   ```bash
   pip install stlite
   ```

## Usage Guide

To run the application:
```bash
stlite run app.py
```
